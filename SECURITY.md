# Security Policy

We take the security of AgentForge seriously. Please review this document to understand supported versions and how to report vulnerabilities.

---

## 1. Supported Versions

We actively provide security patches to the following versions of AgentForge:

| Version | Supported |
| :--- | :--- |
| 0.2.x | Yes (Current release) |
| 0.1.x | No |

---

## 2. Reporting a Vulnerability

If you discover a security vulnerability in this project, **do not open a public issue.** Instead, please report it directly through the following process:

1. **Email us**: Send a detailed report to `security@agentforge.dev`.
2. **Details to include**:
   - Description of the vulnerability.
   - Step-by-step instructions to reproduce the issue.
   - Potential impact (e.g. privilege escalation, remote code execution).
3. **Response timeline**: We will acknowledge your report within 48 hours and work with you to patch the issue before making it public.
